## Adding to User PATH

Use PowerShell to persistently append the folder (e.g., C:\PortablePowerShell) to your user PATH via the registry: `$currentPath = [Environment]::GetEnvironmentVariable('Path', 'User'); $newPath = $currentPath + ';C:\PortablePowerShell'; [Environment]::SetEnvironmentVariable('Path', $newPath, 'User')`.
