## When to Use What

| Approach                    | Use When                                                                                                                    |
| --------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| **Direct API calls**        | You control the code flow, just need research data [docs.perplexity](https://docs.perplexity.ai/getting-started/quickstart) |
| **Gemini Function Calling** | Building autonomous agents that decide when to research [ai.google](https://ai.google.dev/gemini-api/docs/tools)            |
| **MCP Server**              | Want it available in Antigravity IDE/Cursor for interactive use [antigravity](https://antigravity.google/docs/mcp)          |
