# Issue Tracking: Bifrost Bridge

## Logging Requirements

- All bugs and enhancements MUST be logged in `docs/issues-and-enhancements.md`.
- No coding begins without a logged issue/enhancement reference.

## Categories

- **ISS-###**: Bug or technical issue.
- **ENH-###**: Feature enhancement or new capability.

## Resolution

- Link PRs and commits to the issue reference.
- Verify resolution before closing.
