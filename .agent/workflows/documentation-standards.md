# Documentation Standards: Bifrost Bridge

## Requirements

- **Non-Technical Focus**: User guides must be accessible to non-engineers.
- **Principle of Least Astonishment**: Documentation should match actual behavior.
- **Verification**: All documented workflows must be tested/verified.

## Mandatory Docs

- `README.md`: Root overview.
- `STATUS.md`: Continuity tracker.
- `docs/setup_guide.md`: Environment setup.
