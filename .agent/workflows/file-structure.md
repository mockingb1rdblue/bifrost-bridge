# File Structure: Bifrost Bridge

## Directory Organization

- `src/`: Core TypeScript SDK and CLI logic.
- `workers/`: Cloudflare Worker source code (proxies).
- `scripts/`: Python and Shell automation scripts.
- `.agent/`: Agent-specific workflows and skills.
- `docs/`: User and technical documentation.

## Naming Conventions

- **Files**: kebab-case (e.g., `perplexity-client.ts`)
- **Classes**: PascalCase
- **Functions/Variables**: camelCase
- **Constants**: SCREAMING_SNAKE_CASE
